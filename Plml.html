Plml

The Python Library for MetaLinks

Version 0.1, April 2010

This library implements as of now a parser and set of utilities for using Metalinks, an XML-based way of describing how to download and install a set of files.  The only other programs needed are a Python interpreter, lxml, and GNU Privacy Guard for signature functions. <bittorrent> elements are not yet supported, given the lack of support for torrents in the standard library :-) Overall it's lightweight, even with schema checks and file text storage. Given that this is a very new project, there are probably still some major bugs still in the code and any feedback on them would be greatly appreciated.

Goals and Objectives

The intent is to support the entire specification, nonstandard variations, two-way porcessing, and possibly support for revision control systems (like the one you're staring at right now). With performance still in mind, as much of the library should be written in Python as possible to allow for more extending and improvement. Follow the style, share, and enjoy.
